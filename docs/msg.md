Формат хранения сообщений
===

```

# комментарии начинаются с решети
   # у любой строки убираются все пробелы слева и справа
   
:ключи начинаются с двоеточия
всё, что идёт после ключа, является сообщением

количество переносов внутри сообщения сохраняется


# Все пробелы в начале и конце строки удаляются

:ключ 2
...

:ключ 3


```
